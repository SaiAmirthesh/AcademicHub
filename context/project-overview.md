# Project Overview

Project: AcademicHub

## Vision
A university management platform for Students, Faculty, Departments, Courses, Enrollments, Grades and Analytics.

## Roles
- Admin
- Faculty
- Student

## In Scope
Authentication, Departments, Faculty, Students, Courses, Enrollments, Grades, Dashboard, Reports.

## Out Of Scope
Mobile apps, payments, LMS content delivery.

## Success Criteria
All modules operational with PostgreSQL + Drizzle and role-based access.
